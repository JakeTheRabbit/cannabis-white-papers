## How it's built

A small Python build system in `_build/` renders every page through one shared template (theme, nav, glossary, citations, search), and **also generates this README**, `manifest.json`, `llms.txt`, the per-paper Markdown in `papers/`, and the search index.

```
cd _build && python build.py
```

**Add a paper:** create `_build/paper_<slug>.py` (`SLUG, TITLE, EYEBROW, SUB, META, SECTIONS, RELATED, REF_IDS`; `figure()` takes a raw inline `<svg>` string), register it in `build.py` (`PAPER_MODULES`) and `data/nav.py`, add interlink phrases in `links.py`, then rebuild. Citations live in `data/refs.py`.
